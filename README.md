Este trabalho consiste no desenvolvimento de um interpretador abstrato para uma linguagem imperativa (IMP) em Haskell, aplicando os conceitos formais da Semântica Operacional Big-Step. O sistema mapeia e executa de forma puramente funcional estruturas sintáticas de expressões aritméticas, booleanas e comandos complexos de controle. A implementação prioriza a robustez acadêmica ao simular estados de memória mutáveis utilizando estritamente casamento de padrões e recursão estrutural pura.
